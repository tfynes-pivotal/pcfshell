PCF SHELL

Welcome to PCFShell

PCFShell is an administrative "jumpbox as a service", presenting a linux terminal that runs inside any browser. 

Usage:

Select a SHELL instance below: Default username=pcfuser and default password=-------

Be sure to 'setpassword' before using shell to log into any system.

Remember your instance number and type 'logoff' when finished to reset it back to default login and wipe any changes you've made.
